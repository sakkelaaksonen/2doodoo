# 2doodoo

A mimimal library Todo app with Valtio state management

## Tech Stack

- Valtio
- Mustache
- Eleventy
- ESBuild
- QUnit

## Start

    npm run dev
    npm run build
    npm start

## QUnit

- Browser runner at /qunit/
- npm run tests

## Synopsis

- A Todo excercise with Valtio as a reactive state manager. Focus on a18y with minimal libs.
